# AEGIS

AEGIS is my personal research hub for aerospace, physics, simulations, and notes from projects like Project DEUS.  
Think of it as a single place to store ideas, experiments, and learning, all organized and ready to grow.

---

## Structure

- `Papers/` – PDFs, preprints, articles  
- `Simulations/` – Python scripts, models  
- `Notes/` – Personal insights, logs, frameworks  
- `References/` – Citation files, datasets  
- `Resources/` – Helpful links or guides
